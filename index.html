Yammer .NET Client

Welcome! YamNet is an unofficial, opinionated, Yammer REST API .NET (async) wrapper. The goal is to provide a ready-to-use client library available via NuGet, enabling .NET developers to start working with Yammer data quickly.

Learn more »

Simple

Easy to use, easy to understand. Let YamNet take care the bulk of the ceremonial stuff, enabling you to focus on writing nice, clean, readable (async) code. The results are available as strongly-typed DTOs.

Multi-Platform

Taking advantage of Portable Class Library, YamNet is available for use in a wide array of projects, including: .NET 4+, Silverlight 5+, Windows Phone 8+, Xamarin.iOS, and Xamarin.Android. Also available is the .NET 3.5 backport.

Async / Await

Improve responsiveness and scalability. Keep your UI responsive while retrieving data, and enable better scaling for your server-side solution through reduced memory / thread usage, and non-blocking operations.

Get up and running quickly

PM> Install-Package YamNet.Client
